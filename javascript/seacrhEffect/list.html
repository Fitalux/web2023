• padding : 요소의 내부 여백을 설정한다.
  • padding-bottom : 요소의 하단 내부 여백 값을 지정한다.
  • padding-left : 요소의 왼쪽 내부 여백 값을 지정한다.
  • padding-right : 요소의 오른쪽 내부 여백 값을 지정한다.
  • padding-top : 요소의 상단 내부 여백 값을 지정한다.
  • page-break-after : 인쇄될 문서에서 페이지의 마지막 요소의 뒤에 페이지 나누기 삽입 유무의 여부를 지정한다.
  • page-break-before : 인쇄될 문서에서 페이지의 시작 부분의 요소의 앞에 페이지 나누기 삽입 유무의 여부를 지정한다.
  • page-break-inside : 요소 내에서 페이지가 나뉘는 방법을 지정한다.
  • perspective : 3D 변환을 사용하는 요소의 원근을 설정한다. (사용자와 z=0 평면 사이의 거리를 설정)
  • perspective-origin : perspective 속성에 대한 원점을 설정한다. 3D 공간에서 사용자가 보는 위치를 지정한다.
  • pointer-events : 요소가 마우스 이벤트에 응답해야하 하는지의 여부를 지정한다.
  • position : 요소를 위치 지정 방법을 정의한다.
  • quotes : blockquote 또는 p 요소에서 사용될 따옴표를 지정한다.
  • resize : 사용자가 요소의 크기를 조정할 수 있는지의 여부를 지정한다.
  • right : 요소를 수평 방향으로 오른쪽에서부터 위치 시킨다.
  • tab-size : 탭 문자의 너비를 설정한다.
  • table-layout : 테이블을 레이아웃하는 알고리즘을 제어한다.
  • text-align : text-align 속성은 텍스트 정렬 방식을 설정한다.
  • text-align-last : text-align-last 속성은 문단의 마지막 텍스트 정렬 방식을 설정한다.
  • text-decoration : text-decoration 속성은 텍스트 라인 속성을 일괄적으로 설정한다.
  • text-decoration-color : text-decoration-color 속성은 글자 라인의 색상 속성을 설정한다.
  • text-decoration-line : text-decoration-line 속성은 글자 라인의 위치 속성을 설정한다.
  • text-decoration-style : 텍스트 장식 라인(밑줄)의 스타일을 지정하는데 사용된다.
  • text-decoration-skip-ink : text-decoration-skip-ink 속성은 언더 라인의 글씨 겹침을 설정한다.
  • text-decoration-thickness : text-decoration-thickness 속성은 글자 라인의 굵기 속성을 설정한다.
  • text-emphasis-color : text-emphasis-color 속성은 텍스트에 강조 표시 색상을 설정한다.
  • text-emphasis-position : text-emphasis-position 속성은 텍스트에 강조 표시 위치를 설정한다.
  • text-emphasis-style : text-emphasis-style 속성은 텍스트에 강조 표시 스타일을 설정한다.
  • text-emphasis : text-emphasis 속성은 텍스트에 강조 표시를 일괄적으로 설정한다.
  • text-indent : text-indent 속성은 문단 들여쓰기 속성을 설정한다.
  • text-justify : text-justify 속성은 양쪽 정렬을 했을 때 정렬 유형을 설정한다.
  • text-overflow : text-overflow 속성은 텍스트가 영역을 벗어 났을 때 속성을 설정한다.
  • text-shadow : text-shadow 속성은 텍스트 그림자를 설정한다.
  • text-transform : 텍스트의 대문자와 소문자를 변환하는데 사용된다.
  • text-underline-offset : text-underline-offset 속성은 언더 라인의 오프셋 값을 설정한다.
  • text-underline-position : text-underline-position 속성은 언더 라인의 위치를 설정한다.
  • top : 요소가 위치한 요소 상단과 포함하는 블록 상단 간의 거리를 지정한다.
  • transform : 요소를 2D 또는 3D 공간에서 변형할 수 있게 하는 속성이다.
  • transform-origin : 변환의 기준점을 설정하여 변형이 적용되는 위치를 결정한다.
  • transform-style : 변형된 요소의 하위 요소가 하나의 평면으로 펼쳐져야 하는지 아니면 3D 위치를 유지해야 하는지를 지정한다.
  • transition : 속성이 두 상태 사이에서 어떻게 전환되어야 하는지를 지정한다
  • transition-delay : 전환이 트리거 된 시간과 시작 시간 간의 지연 시간을 지정한다.
  • transition-duration : 전환을 완료하는데 소요되는 시간을 지정한다.
  • transition-property : 전환할 속성을 지정한다.
  • transition-timing-function : 전환에서 사용되는 이징 함수를 지정하여 시간에 따른 변경 비율을 결정한다.
  • unicode-bidi : 텍스트의 방향성을 컨텐츠 부모 요소에 의해 결정해야 하는지를 지정한다.
  • user-select : 사용자가 텍스트를 선택할 수 있는지의 여부를 제어한다.
  • vertical-align : 인라인 요소 또는 테이블 셀의 수직 정렬을 지정한다.
  • visibility : 요소가 보이거나 숨겨져 있는지를 지정한다.
  • white-space : white-space속성은 줄바꿈 및 공백을 설정한다.
  • width : width 속성은 요소의 가로 크기를 설정한다.
  • word-break : word-break 속성은 줄바꿈을 할때 단어를 기준으로 설정하는 속성이다.
  • word-spacing : 단어 간 간격을 지정한다.
  • word-wrap : 요소의 내용이 요소의 경계를 벗어날 때 줄바꿈이 발생해야 하는지 여부를 지정한다.
  • z-index : 요소의 쌓임 순서를 지정한다. 값이 높을수록 요소의 맨 위에 쌓인다.
